# DevLinks — Pedro Aruanã

Projeto final do curso **Discover** da Rocketseat. Uma página de links personalizada com tema dark/light, animações e seção de projetos.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Alternância entre tema escuro e claro
- Efeito de digitação no terminal
- Copiar e-mail ao clicar
- Animações de entrada e hover nos cards
- Layout responsivo para mobile

## Como acessar

Basta abrir o arquivo `index.html` no navegador.
